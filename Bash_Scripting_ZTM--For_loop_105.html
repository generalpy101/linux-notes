For loop


For loop is a looping statement. It is used t operform a task repeadately over some sequence.

Here sequence can be a list of strings separated by a space, range of numbers, output of a command, an array and so on.

Syntax of for loop is :

for item in sequence
do
    #commands
done

Let's look at some examples.

In the above example we gave names of os as a sequence. Strings separated by space are regarded as list and if we want to include a space separated string in it enclose it in quotes.

In above example we created a range of numbers. Syntax to create range of numbers is :

{start..stop..increment}

We can also create C style loops in bash tho they are seldom used.

Enclose C style loop conditions inside double parentheses and you are good to go.

For loops can also operate over commands.
