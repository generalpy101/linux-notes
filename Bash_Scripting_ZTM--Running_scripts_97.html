Running scripts


There are multiple ways to run a bash script.

Simplest way is the way we've been using since since last 2-3 lectures.

./script_name or absolute_path_to_script

When running script this way, execute permission should be there and should be appropriate shebang line.

Second method to run shell scripts is to actually use the interpreter. Examples :

python3 python_file
bash bash_file

This requires no permissions and ovverides the shebang line.

Third way is to use source command or .

source file
. file

Both methods are same .

The difference between first and third method is that first method runs command inside a new shell or sub shell while third method runs commands in the current shell.
