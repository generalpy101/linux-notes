Step 4 : Quote Removal


The shell will remove from the final command any unquoted \, ‘’ and “” that didnot result from a shell expansions.

For example :

echo \$HOME

This will print $HOME on screen, the backslash was a quote and was removed.

echo '\$HOME'

This will print \$HOME as ‘’ treat everything as a string literal hence single quotes are removed but not inner backslash.
