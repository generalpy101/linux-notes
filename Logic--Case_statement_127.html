Case statement


Case statement can be used as an alternative to if statements in some cases.

Case statment takes a variable and checks it against multiple patterns to provide conditionals.

case variable in
    pattern) #commands ;;
    pattern) #commands ;;
    ...
    ...
esac

Variable is there which is to be compared and each pattern match must end with ;;. Each pattern must end with ).

Default condition is added using * to match everything. Generally used as a default case.
