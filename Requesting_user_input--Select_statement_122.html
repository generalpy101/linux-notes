Select statement


Select statement is used to easily create menu from a list.

Syntax of select is :

select var in list 
do
    #commands
done

This prompts a menu with all options in the list numbered and provide user with a prompt where user's input is stored is REPLY variable. If user inputs nothing, menu is shown again with the prompt. The value is user selected reply is stored in the variable var. Select keeps on executing itself till break or ctrl + c is pressed.

The prompt of select is governed by the variable PS3.
