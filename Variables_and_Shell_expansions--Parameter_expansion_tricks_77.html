Parameter expansion tricks


We can easily manipulate values stored inside variables while displaying.

To perform these manipulations, we must use {} expansion.

Add , at the end of variable name to make first letter of the variable value small case.

name="GeNeralpy"
echo ${name,}

Add ,, at the end of variable name to make whole value of variable in small case.

echo {name,,}

Note that actual values of variables are not changed, they are just displayed in a different manner.

Add ^ at the end of the variable name to make first letter capitalized.

echo {name^}

Add ^^ at the end of variable name to make whole value upper case.

echo {name^^}

To count the size of variable's value(length of variable value string) user # before the variable name .

echo {#name}

We can slice a string by below syntax :

${variable:offset:length}

offset is starting location.String numbering start from 0.

We can specify negative offset too where numbering will start from the end with last element being -length. When using negative indexing, add space before the index.
