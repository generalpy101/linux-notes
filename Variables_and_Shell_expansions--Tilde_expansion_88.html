Tilde expansion


Tilde(~) by default returns the currrent user's home directory. It is basically showing value of $HOME variable. If we change value of $HOME value returned by ~ will also change.

~ can also be used for many other tasks.

~ immediately followed by username will give home directory of that user if it exists.

~username

bash stores current directory and previous directories in shell variables $PWD and $OLDPWD. pwd command just prints contents of $PWD. ~+ will return value of $PWD and ~- will return value of $OLDPWD.

~ expansion is useful when working with multiple directories.
