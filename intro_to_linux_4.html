intro to linux


Linux :- Linux is a family operating systems which is based on a kernel(main part of an OS) named Linux created by Linus Torvalds in 1991 as an Open Source alternative to UNIX.

Linux Distribution :- Since Linux kernel is open source, many different types of Linux are there i.e different flavors of Linux OS which usually consists of a software suite, the main linux kernel and often a package manager.

A linux os comprises :

• The main Linux Kernel

• The GNU shell utilities

• The graphical desktop environment and some more utilitiesLinux distributions are not made by one organisation like in case of Windows and Mac, different parts are created by different people.

Some examples of linux distros are Red Hat, Debian, CentOS etc.

EXTRA :- Distrowatch is a good site that keep track of different Linux Distros.

GNU utilities :- GNU shell utilities are some programs which run on shell(terminal) which are core linux components used for daily tasks like copying files, administrator rights management etc.a
