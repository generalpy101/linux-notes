some useful keyboard shortcuts


Ctrl + A : Brings cursor to beginning of the current line

Ctrl + D : Closes the current shell same as exit command

Ctrl + L : Clears the terminal screen same as writing clear command

Ctrl + E : Moves the cursor to the end of the line

Ctrl + U : Cuts everything before the cursor. Useful when a long command is needed to be erased.

Ctrl + Y: Paste text which was previously cut.

Ctrl + C : Suspends any running process/command. Is used to completely stop a command in between(like ping command which keeps on running if not stopped) or to stop a process in between like downloading using wget.

Ctrl + Z : Interrupts any running process/command. Unlike previous command which completely stops the program it temporarily suspends it and that command can be resumed by using some commands(more on this later).

Up and Down arrow : They are used to navigate between previous commands.
